BUAA 2023 Autumn OOP

This is the source code of the blog for BUAA 2023 autumn Object-Oriented Course.

Before You Start

Make sure you modify the deploy options in _config.yaml!

If you want to use hx.bat, remember to replace the directory to yours.

Add Article

Draft layout is recommended.

hexo new draft "title"
# edit article...
hexo publish draft "title"

Local Server & Deploy

A simple batch file hx.bat is provided.

hx.bat server    # hexo generate && hexo server
hx.bat server -c # hexo clean && hexo generate && hexo server
hx.bat deploy    # hexo generate && hexo deploy
hx.bat deploy -c # hexo clean && hexo generate && (do patch) && hexo deploy
